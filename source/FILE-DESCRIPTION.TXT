For manual compilation with gfortran it is recommended you use the debug flags
located in: GFORTRAN-DEBUG-FLAGS.TXT. The program has not be compiled with any
other compilers than GNU gfortran but it is built with no known compiler dependencies.
inputFile.xml is a user defined language you can import into Notepad++ to
provide mild syntax highlighting on the model's *.INP files.

Files compiled for Aperture Map Model:
      - APERTURE_MAP_FLOW.F
      - APM_FLOW.F
      - APM_MODULE.F
      - APM_OUTPUT.F
      - APM_SOLVER.F
      - APM_SUBROUTINES.F
      - UNIT_CONVERSION_MODULE.F
